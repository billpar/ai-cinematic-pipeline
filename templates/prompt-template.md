# Prompt Template — One Beat

Use this structure for every single generation prompt. Fill each bracket, delete instructions when done.

```
[TIMESTAMP, e.g. 00:00 - 00:02]

[SHOT TYPE] shot, [LENS]mm, [APERTURE].
[CAMERA BEHAVIOR — static / slow push / handheld / pan — must match the emotional beat]

@[CharacterTag] [physical action — what the body is doing right now, not what it means]
[Position: sitting/standing, facing direction, distance from other characters in meters]
[Expression: physical micro-muscle detail, never an emotion word]

[If dialogue present: Character says, "exact line." — describe pre-line beat, during-line beat, post-line beat]
[Do NOT include voiceover lines here — those go to your voice tool separately]

[Audio line: ambient sound, specific sound effects, NOT music]

@[SettingTag]
```

## Filled-in example (generic placeholder — replace with your own content)

```
[00:14 - 00:17]

Medium close shot, 85mm, f/1.8.
Camera holds static, no movement.

@Character_A sits facing a central object, hands engaged in a small
repetitive task. They do not look toward the doorway. Expression:
right corner of mouth lifts slightly, eyes unchanged.

Character A says, "Line of dialogue here." Beat before: hands keep
working, no pause. Beat after: they resume immediately, no reaction shift.

Audio: a specific mechanical or ambient sound, faint background noise.

@Location_A
```

## Checklist before submitting

- [ ] One camera instruction only
- [ ] One action only
- [ ] One lighting note only (and it matches the locked setting description)
- [ ] Expression is physical, not emotional
- [ ] Spatial blocking declared if 2+ characters
- [ ] Only elements that appear in this beat are attached
- [ ] No voiceover text included
- [ ] No aspect ratio mentioned in the text
- [ ] No negative prompts
